> CLI for listing and filtering user npm packages from the [npmjs.com](https://npmjs.com)!  Using [npm-pkgs-filter](http://j.mp/1J8UJe3)

## Install
```
npm i npm-pkgs-cli -g
```


## CLI
> run `npm-pkgs --help` or try this one

```
npm i -g npm-pkgs

  Aloha, master!

  ℹ Please wait a moment...
```
